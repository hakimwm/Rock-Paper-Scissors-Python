# Interactive Rock, Paper, Scissors Game

A clean, terminal-based implementation of the classic Rock, Paper, Scissors game built using Python. The project focuses on structured functional programming, robust user input handling, and automated conditional logic.

## 🛠️ Technologies Used
* **Language:** Python 3.x
* **Core Modules:** `random` (Standard Library)

## 🚀 Key Implementation Features
* **Functional Architecture:** Separately modularized input collection (`get_choices`) and structural evaluation logic (`check_win`) to keep the code organized and easy to maintain.
* **Case-Insensitive Input Handling:** Utilizes advanced string manipulation via `.casefold()` to ensure that user inputs are processed accurately regardless of casing variations (e.g., matching "Rock", "ROCK", or "rock" perfectly).
* **Automated Opponent Selection:** Integrates Python's pseudo-random number generation capabilities to simulate a real-time computer opponent picking moves dynamically from data lists.
* **Engaging User Feedback:** Formatted terminal outputs with responsive conditional outcomes and illustrative icons/emojis to elevate the standard user interface experience.

## 📂 Project Structure
* `rps_game.py` - The primary source code containing game states, logic loops, and operational functions.
